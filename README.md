# Bangladesh-Themed Portfolio Website

This repository contains my personal portfolio website built with HTML, CSS, and JavaScript for **Solo Sprint 2**.  
The site is themed around Bangladesh (colors + storytelling) and includes a small Three.js 3D element in the hero.

## Live Site

After deployment on GitHub Pages:

`https://princessibtihaj.github.io`

## Project Highlights

- Bangladesh-inspired visual design and color palette
- Responsive single-page portfolio layout
- Experience timeline and project highlights
- Embedded YouTube showcase speech
- Interactive "Journey" boat section (light gamification)
- Three.js 3D Bangladesh-inspired model in the hero
- Contact form that opens an email draft (`mailto`)

## Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- [Three.js](https://threejs.org/) (via CDN)

## Run Locally

### Open directly

```bash
open index.html
```
